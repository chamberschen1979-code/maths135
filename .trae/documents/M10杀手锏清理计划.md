# M10 杀手锏清理计划（修订版）

## 一、问题分析

### 当前状态

* M10共有75种strategy\_hint

* weapon\_details.json中CONIC（圆锥曲线）只有9个杀手锏

* M01-M06每个母题的杀手锏数量：3-10个

### weapon\_details.json 杀手锏分布参考

| 类别              | 杀手锏数量  |
| --------------- | ------ |
| QUAD(二次函数)      | 1个     |
| SET(集合)         | 1个     |
| LOGIC(逻辑)       | 2个     |
| COMPLEX(复数)     | 2个     |
| LOG(对数)         | 3个     |
| TRIG(三角)        | 4个     |
| FUNC(函数)        | 5个     |
| GEO(立体几何)       | 5个     |
| DERIV(导数)       | 5个     |
| VEC(向量)         | 6个     |
| TRI(解三角形)       | 8个     |
| SEQ(数列)         | 9个     |
| **CONIC(圆锥曲线)** | **9个** |
| INEQ(不等式)       | 10个    |

### 结论

**M10应该保留5-9个杀手锏，与CONIC的9个杀手锏对应**

## 二、weapon\_details.json 中已有的CONIC杀手锏

| ID         | 名称            | 核心逻辑                    |
| ---------- | ------------- | ----------------------- |
| S-CONIC-01 | 定义降维与第三定义     | 斜率积为定值±b²/a²            |
| S-CONIC-02 | 焦点三角形面积公式     | S=b²tan(θ/2)或b²cot(θ/2) |
| S-CONIC-03 | 渐近线几何性质与距离公式  | 焦点到渐近线距离=b              |
| S-CONIC-04 | 几何不等式构造与边界分析  | 几何约束翻译为代数不等式            |
| S-CONIC-05 | 仿射变换          | 椭圆压成圆，面积比=b/a           |
| S-CONIC-06 | 齐次化联立         | 处理斜率关系问题                |
| S-CONIC-07 | 参数方程          | 椭圆上点设为(acosθ, bsinθ)    |
| S-CONIC-08 | 阿波罗尼斯圆（阿氏圆）性质 | 距离比为定值λ的轨迹是圆            |
| S-CONIC-09 | 等轴双曲线性质       | a=b, e=√2               |

## 三、M10 strategy\_hint 清理方案

### ✅ 保留（与weapon\_details对应）- 9个

| strategy\_hint    | 对应杀手锏      | 理由                       |
| ----------------- | ---------- | ------------------------ |
| **阿波罗尼斯圆（阿氏圆）性质** | S-CONIC-08 | 已有，核心杀手锏                 |
| **等轴双曲线性质**       | S-CONIC-09 | 已有，特殊模型                  |
| **第三定义直接应用**      | S-CONIC-01 | 已有，公式秒杀                  |
| **焦点三角形面积公式**     | S-CONIC-02 | 已有，公式秒杀                  |
| **渐近线夹角公式**       | S-CONIC-03 | 已有，公式秒杀                  |
| **几何不等式构造**       | S-CONIC-04 | 已有，技巧转化                  |
| **内切圆半径公式**       | 新增         | 公式秒杀，建议加入weapon\_details |
| **角平分线定理与定义**     | 新增         | 技巧转化，建议加入weapon\_details |
| **焦点弦性质**         | 新增         | 模型识别，建议加入weapon\_details |

### ❌ 删除（非杀手锏）- 66个

**删除类型1：纯方法描述**

* 坐标代入法、点到直线距离、距离公式直接列式

* 直接应用性质、渐近线与参数关系

* 平行条件与斜率、垂直关系与斜率

**删除类型2：标准解题步骤**

* 设直线方程与韦达定理、焦半径公式与韦达定理

* 分类讨论与定义结合、分类讨论与勾股定理

* 定义联立与面积法、设斜率消参法

**删除类型3：过于宽泛**

* 位置关系分析、定义与不等式分析

* 几何性质与三角函数、对称性与角度分析

**删除类型4：重复/相似内容**

* 面积公式直接应用、面积公式与存在性分析 → 合并到"焦点三角形面积公式"

* 内切圆半径公式与最值、内切圆性质与定义结合 → 合并到"内切圆半径公式"

* 夹角公式与半角、夹角与斜率关系 → 合并到"渐近线夹角公式"
