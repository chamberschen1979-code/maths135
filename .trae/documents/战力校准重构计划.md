## 核心算法：分级门槛制

### 1. 天花板判定 (Ceiling Logic)

```javascript
const getEloCeiling = (subTargets) => {
  if (!subTargets || subTargets.length === 0) return 2000
  
  // 检查 L2 级变例
  const l2Subs = subTargets.filter(sub => sub.level_req === 'L2')
  const hasUnmasteredL2 = l2Subs.some(sub => !sub.is_mastered)
  if (hasUnmasteredL2) return 1199
  
  // 检查 L3 级变例
  const l3Subs = subTargets.filter(sub => sub.level_req === 'L3')
  const hasUnmasteredL3 = l3Subs.some(sub => !sub.is_mastered)
  if (hasUnmasteredL3) return 1599
  
  // 检查 L4 级变例
  const l4Subs = subTargets.filter(sub => sub.level_req === 'L4')
  const hasUnmasteredL4 = l4Subs.some(sub => !sub.is_mastered)
  if (hasUnmasteredL4) return 1799
  
  return 2000  // 全部掌握，无上限
}
```

### 2. 动态计分 (Scoring Logic)

```javascript
const calculateEloFromSubTargets = (subTargets) => {
  if (!subTargets || subTargets.length === 0) return 800
  
  let elo = 800  // 初始分（L1 状态）
  
  // 基础加分：每点绿一个 L2 变例，Elo +100
  subTargets.forEach(sub => {
    if (sub.is_mastered && sub.level_req === 'L2') {
      elo += 100
    }
  })
  
  // 进阶加分：每点绿一个 L3 变例，Elo +150
  subTargets.forEach(sub => {
    if (sub.is_mastered && sub.level_req === 'L3') {
      elo += 150
    }
  })
  
  // 封神加分：每点绿一个 L4 变例，Elo +200
  subTargets.forEach(sub => {
    if (sub.is_mastered && sub.level_req === 'L4') {
      elo += 200
    }
  })
  
  // 应用天花板
  const ceiling = getEloCeiling(subTargets)
  return Math.min(elo, ceiling)
}
```

### 3. 计分示例

**场景：某母题有 3 个变例（L2, L3, L4 各一个）**

| 掌握状态 | 计算 | ELO | 说明 |
|---------|------|-----|------|
| 全红 | 800 | 800 | L1，天花板 1199 |
| L2绿 | 800 + 100 = 900 | 900 | L1，天花板 1599 |
| L2+L3绿 | 800 + 100 + 150 = 1050 | 1050 | L2，天花板 1799 |
| L2+L3+L4绿 | 800 + 100 + 150 + 200 = 1250 | 1250 | L2，无上限 |
| 只 L4 绿 | 800 + 200 = 1000 | **1199** | 被天花板限制！L2有红 |

**关键点：L4 绿了但 L2 红着，分数被锁在 1199！**

### 4. 等级判定

| ELO 范围 | 等级 | 健康状态 |
|---------|------|---------|
| 800-999 | L1 | bleeding |
| 1000-1299 | L2 | bleeding |
| 1300-1699 | L3 | bleeding |
| 1700+ | L4 | healthy |

   ```javascript
   const handleRecalculateElo = (targetId) => {
     setTacticalData(prevData => {
       const newData = { ...prevData }
       for (const map of newData.tactical_maps) {
         const encounter = map.encounters.find(e => e.target_id === targetId)
         if (encounter) {
           encounter.elo_score = calculateEloFromSubTargets(encounter.sub_targets)
           encounter.gear_level = getLevelByElo(encounter.elo_score)
           encounter.health_status = encounter.elo_score >= 1700 ? 'healthy' : 'bleeding'
           break
         }
       }
       return newData
     })
   }
   ```
