# M01-M17 变例因子权重分析计划

## 一、分析目标

遍历 M01-M17 所有 JSON 文件，完整列举每个母题-专项-变例的变例因子（variable_knobs）及其权重（weight），并分析权重的合理性。

## 二、权重合理性评判标准

| 层级 | 权重分布原则 | 校验要点 |
|------|-------------|---------|
| **L2（基础）** | weight 应高度集中在基础项，冷门/困难项权重应设为 0 或极低 | 基础项权重 ≥ 0.6，困难项权重 ≤ 0.2 |
| **L3（进阶）** | weight 应均衡分布，涵盖主流考点和典型陷阱 | 各项权重差异不宜过大（0.3-0.5 区间） |
| **L4（巅峰）** | weight 应向高难度构造和复合陷阱倾斜 | 高难度项权重 ≥ 0.5，基础项权重 ≤ 0.3 |
| **总和校验** | 同一类别下所有 weight 之和理论上应为 1.0 | Σweight = 1.0（允许 AI 自动归一化） |
