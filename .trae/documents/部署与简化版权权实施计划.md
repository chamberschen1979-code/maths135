# 高中数学无忧 - 部署与简化版权权实施计划

## 目标
1. 部署项目到 Zeabur 公网访问
2. 添加简化版权权管理（管理员密码+查看所有用户）

## 优先级
1. **① 部署 Zeabur** - 今天就能让人试用
2. **② 简化版权权** - 管理员密码解锁全部功能
3. **③ 数据库** - 暂缓，localStorage对几个人试用够用

---

## 步骤一：Zeabur 部署

### 1.1 代码推到 GitHub
- [ ] 创建 GitHub 仓库（如果还没有）
- [ ] git init, git add, git commit
- [ ] git remote add origin, git push

### 1.2 Zeabur 部署
- [ ] 访问 zeabur.com，用 GitHub 登录
- [ ] 点击 "New Project" → "Deploy from GitHub"
- [ ] 选择仓库，选择分支
- [ ] Zeabur 自动检测 Vite 项目，一键部署
- [ ] 获取公网 URL（如 `xxx.zeabur.app`）

### 1.3 验证
- [ ] 用手机/另一台电脑访问，确认正常

---

## 步骤二：简化版权权管理

### 2.1 设计方案
- 管理员有一个密码（如 `admin2024`）
- 输入正确密码后，可解锁：
  - 查看所有用户的学习进度
  - 重置/删除用户数据
- 普通用户：
  - 直接使用，不需要登录

### 2.2 实现文件修改

#### 2.2.1 创建管理员上下文 `src/context/AdminContext.jsx`
- 管理管理员状态
- adminPassword: 简单密码验证
- isAdmin: boolean
- login(password): 验证密码，解锁admin模式
- logout(): 退出admin模式

#### 2.2.2 创建管理员面板组件 `src/components/admin/AdminPanel.jsx`
- 查看所有用户列表（用户名、最后登录时间、ELO概览）
- 查看指定用户的详细学习进度
- 删除/重置用户数据

#### 2.2.3 修改 `App.jsx`
- 添加"管理员入口"按钮（在首页角落，小图标）
- 点击输入密码，解锁后显示 AdminPanel
