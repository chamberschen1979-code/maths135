# App.jsx 拆分计划

## 拆分目标

将 `App.jsx`（2063行)拆分为更小的模块，提高代码可维护性。

## 拆分原则

1. **按功能模块拆分**：相关功能放在一起，便于维护
2. **按数据流拆分**：状态管理、API调用、业务逻辑分离
3. **按组件拆分**：大型组件独立成文件
4. **按工具函数拆分**：纯函数放到独立文件

5. **按类型拆分**：Type定义和常量移到独立文件

6. **按常量拆分**：配置、常量移到独立文件

7. **按样式拆分**：主题相关样式移到独立文件

8. **按 Context 拆分**：ThemeContext 和 GradeContext 移到独立文件

9. **按 Hooks 拆分**：自定义 Hooks 移到独立文件

10. **按组件拆分**：大型组件移到独立文件
11. **按初始化模态框拆分**：初始化相关的 UI 和逻辑移到独立文件
