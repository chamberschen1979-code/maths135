# 杀手锏分级规划方案（实战版）

## 核心原则

**一切从高考实战出发，宁缺毋滥，允许跨模块匹配**

| 原则 | 说明 |
|------|------|
| ✅ 实战有用 | 题目确实需要这个方法才能快速解决 |
| ✅ 宁缺毋滥 | 没有合适的就不匹配，不要硬塞 |
| ✅ 允许跨模块 | M04 题目可以用函数、导数杀手锏 |
| ✅ 0-2 个 | 每道题匹配 0、1 或 2 个杀手锏 |

## 四、匹配规则（实战版）

### 4.1 核心逻辑

```javascript
// 匹配原则：实战有用，宁缺毋滥
function matchWeapons(problem, answer, keyPoints, analysis) {
  const content = `${problem} ${answer} ${keyPoints} ${analysis}`;
  const matchedWeapons = [];
  
  // 遍历所有 Level 2+ 杀手锏
  for (const [weaponId, weapon] of weaponDetails) {
    // 检查是否实战有用
    const isPractical = checkPracticalUse(content, weapon);
    if (isPractical) {
      matchedWeapons.push(weaponId);
    }
  }
  
  // 宁缺毋滥：只取前 2 个最相关的
  return matchedWeapons.slice(0, 2);
}

// 检查是否实战有用
function checkPracticalUse(content, weapon) {
  // 1. 必须匹配 ≥2 个关键词
  const keywordMatches = weapon.keywords.filter(k => content.includes(k));
  if (keywordMatches.length < 2) return false;
  
  // 2. 排除基础概念（Level 1）
  if (weapon.level === 1) return false;
  
  // 3. 确实能帮助解题
  return true;
}
```

### 4.2 关键词匹配阈值

| 条件 | 要求 |
|------|------|
| 关键词匹配 | ≥ 2 个 |
| Level 要求 | Level 2 或 Level 3 |
| 数量限制 | 0-2 个 |
